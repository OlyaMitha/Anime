Стихия Огня (яп. 火遁 Ка:тон) — один из базовых природных типов чакры. Огонь высвобождается после формирования чакры в желудке, после выпуска её через лёгкие, затем через рот.

Стихия Ветра (яп. 風遁 Фу:тон) — один из пяти основных элементов преобразования природы чакры. Сродство к свойству ветра очень редко, поэтому существует мало техник ветра. Стихия ветра стоит над стихией молнии и под стихией огня, что делает его эффективным против техник молний, но уязвимым к техникам огня. Специальное качество свойства ветра это обладание экстремально острым режущим качеством при правильно выполненной стихии ветра.

Стихия Молнии (яп. 雷遁 Рай:тон) — один из пяти базовых природных типов чакры.

Техники Стихии Молнии позволяют пользователю создавать молнии путём увеличения вибрации высоких частот своей чакры.

Стихия Земли (яп. 土遁 До:тон) — один из пяти основных элементов преобразования природы чакры.

Стихия Воды (яп. 水遁 Суй:тон) — один из пяти основных элементов преобразования природы чакры.

Чакра (яп. チャクラ, Чакура) — компонент, имеющий важное значение для выполнения большинства базовых техник. С помощью различных методов, наиболее распространёнными из которых являются ручные печати, чакру можно контролировать и управлять ею с целью создания некоторого сверхчеловеческого эффекта, достижение которого без этих методов, в противном случае, было бы невозможно. Это касается таких способностей, как хождение по водной поверхности, выдыхание огня из груди, а также создание иллюзий. Обычно чакра не видна невооружённому глазу, за исключением чакры высокой концентрации или проявления её в больших количествах.
